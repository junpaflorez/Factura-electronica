# Final Ing Soft III

El final se compone de 5 puntos.

1. Generar una HU (historia de usuario), esta historia de usuario, busca que se cree un archivo en GIT, con los nombres de los compañeros (TODOS los del salon) (nota: no olvidar la epica)
1. Generar un repositorio en GIT (Github o gitlab) y conectarse (git clone)
1. Generar un branch en el cual, indican como crear la HU en DevOps.
1. Subir la rama junto con el archivo de descripcion de DevOpS
1. A la HU, documentar todo el proceso.

> Comandos GIT:
> - git clone (clona el git o lo descarga)
> - git status (muestra si existen cambios sin guardar o los estados que tienen los archivos)
> - git pull (baja los cambios)
> - git branch (indica en que rama estoy parado)
> - git checkout -b <<feature>> (crea una rama)
> - git checkout <<feature>> (me muevo a una rama)
> - git push (empuja en la rama principal)
> - git remote push origin/<<FEATURE>> (empuja en la rama deseada)
